# \_sto-supported-scanners

If you use a scanner that isn't listed in the following table, you can still ingest your scan results into STO.

* If your scanner can publish to SARIF format, go to [Ingest SARIF scan results into STO](../../use-sto/orchestrate-and-ingest/ingest-sarif-data/).
* For other scanners, go to [Ingest results from unsupported scanners](../../use-sto/orchestrate-and-ingest/ingesting-issues-from-other-scanners.md).
